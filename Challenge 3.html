Programming Languages for Web Development

Web Development is an aspect in software engineering where a website is been built, there are different programming languages that are used in developing a website. these languages work to ensure the smooth flow of events and activities in the website

Website development can be divided into two parts namely ;

  1. Frontend Development
  2. Backend Development

Frontent Development

This entails coding of the part of the website which users interact with, usally involves iinks, buttons, images, tables, dropdowns, forms, etc

The programming languages used in frontend development include :

  1. HTML
  2. CSS
  3. JavaScript

Backend Development

This entails coding of the part of the website which saves, control, and command the instruction given by the uder from the frontend. This deals with data base, data recovery and data entry

The programming languages used in backend development include :

  1. PHP
  2. JavaScript
  3. Python
